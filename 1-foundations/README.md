This module covers the core system administration, networking, scripting, and version control foundations required for modern DevOps engineering.

## 🛠️ Key Foundations Learned
* **The Linux Shell**: System navigation, file manipulations, log inspection, and virtual filesystems (`/proc`, `/sys`).
* **Identity & Access Management (IAM)**: Operating-system level users, groups, sudo privileges, and secure file permission models (e.g. `chmod 600`).
* **Service Management**: Managing web applications (Nginx) using `systemd` daemon controls, unit files, and timers.
* **Log Rotation & Maintenance**: Preventing disk starvation using `logrotate` schedules and writing diagnostic disk report systems.
