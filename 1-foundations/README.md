## 🛠️ Key Foundational Skills
* **System Administration**: Operating inside Ubuntu LTS systems, auditing hardware resources, managing permissions, and configuring daemon managers (systemd).
* **Network Auditing & Diagnostics**: Inspecting interfaces, calculating CIDR subnets, troubleshooting DNS, and configuring secure port-forwarded access pathways.
* **Shell & Code Automation**: Creating custom shell scripts and Python programs to read configuration manifests and query endpoints programmatically.
* **Collaborative Workflows**: Enforcing branch protection rules, signing commits, and setting up repository-level linting hooks.
